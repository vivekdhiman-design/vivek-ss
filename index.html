HTML + JavaScript Tutor: Adding Three Numbers

Step 1: Understand the Goal

In this lesson, we will learn how to add three numbers using JavaScript inside an HTML page.

We will use three numbers: 5, 10, and 15. Our goal is to add them and show the answer on the page.

Step 2: The JavaScript Code

This function takes three values, a, b, and c, then returns their sum. 

function addThreeNumbers(a, b, c) {
  return a + b + c;
}

Step 4: What Happens?

When you click the button, JavaScript runs the function, adds all three numbers together, and displays the answer on the page.
